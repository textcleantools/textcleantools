How to use this tool

  1. Paste a list with one item per line.
  2. Click Sort Lines.
  3. Copy the alphabetized output.

Common uses

  • Alphabetize names, tasks, product lists, or keywords.
  • Prepare lists before pasting into documents.
  • Clean copied spreadsheet rows.

Frequently asked questions

Does it sort numbers naturally?

Yes. The sorter uses numeric-aware sorting where possible, so item 2 sorts before item 10.

Does it remove blank lines?

Yes. Empty lines are removed before sorting.

Is my list uploaded?

No. Sorting is performed in your browser.
