Frequently asked questions

Does it find links that start with www?

Yes. It detects common links starting with http, https, or www, plus many plain domain-style links.

Does it remove duplicate links?

Yes. Duplicate URLs are removed while preserving the first occurrence.

Can ads read the URLs I paste?

No. Advertising cookies do not give Text Clean Tools or advertising partners access to the text entered into the tool.
